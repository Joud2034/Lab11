# Lab11

What the Notebook Includes

Loading and conducting an initial exploration of the credit card customer dataset
Examining the dataset's shape, feature information, and descriptive statistics
Removing the CUST_ID column from the feature space
Identifying and handling missing values through mean imputation
Performing exploratory data visualization to understand feature distributions
Standardizing the features using StandardScaler to prepare data for distance-based clustering
Applying the K-Means clustering algorithm to segment customers
Employing the Elbow Method to determine an appropriate number of clusters
Evaluating clustering quality using the Silhouette Score
Reducing dimensionality with PCA to enable visual interpretation of cluster assignments
Analyzing and interpreting the resulting customer segments
